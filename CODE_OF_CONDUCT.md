# Code of Conduct

## Our Commitment

We want the a11ymind project to be a respectful, constructive place for
discussion and collaboration.

Contributors, maintainers, and participants are expected to help create an
environment that is professional, inclusive, and focused on useful technical
work.

## Expected Behavior

- Be respectful and direct
- Assume good intent, but be clear about problems
- Give actionable feedback
- Keep discussions relevant to the project
- Accept that maintainers may make product and roadmap decisions that differ
  from your preference

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Discriminatory or hateful language
- Doxxing or sharing private information without permission
- Spam, trolling, or intentionally disruptive behavior
- Repeated bad-faith arguments after a moderation request to stop

## Scope

This Code of Conduct applies to:

- GitHub issues
- pull requests
- code review discussions
- project-related community interactions

## Enforcement

Project maintainers may remove comments, close threads, reject contributions,
or block participants whose behavior makes collaboration unsafe or
unproductive.

## Reporting

For conduct concerns related to this repository, contact:

- support@a11ymind.ai

If the report is security-related, use the private reporting guidance in
[SECURITY.md](SECURITY.md).
